# Overall idea
3 + 4 bit adder/multiplier

# Specifications
## __*Inputs:*__ 
    
[0:3]- First number (3 bit number)

[4:6]- Second number (4 bit number)

[7]- High/low -> Addition/multiplication

## __*Outputs:*__ 

[0:7]- binary output (8 bit number)

__*OR*__

Seven-segment display: One digit at a time. Can be reset with input 6
